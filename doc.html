Ethereum Bulk sender an EVM wallet where you can bulk send tokens.

Ethereum bulk sender is an airdrop designed for users who want to airdrop their tokens or send tokens to multiple users in two transactions, with customizable transaction fees.

This Dapps is build using NextJs

What blockchains does Ethereum bulk sender support?

  • Ethereum mainnet
  • Ethereum Sepolia testnet
  • Binance Smart Chain
  • Coinbase Base
  • If you need support for another blockchain, please contact us.

What you can do with Ethereum bulk sender?

  • Airdrop Native Tokens: Utilize bulk sender functionality to optimize transaction fees when distributing native tokens.
  • Airdrop ERC20 Tokens: Employ bulk sender functionality to optimize transaction fees when distributing ERC20 tokens.
  • Airdrop ER721 Tokens: Employ bulk sender functionality to optimize transaction fees when distributing ERC721 tokens.
  • Airdrop ER1155 Tokens: Employ bulk sender functionality to optimize transaction fees when distributing ERC1155 tokens.
  • Add Vip Users to use the service for free.
  • Check your repostring via transactions history.

There are two code sources:

  • Web bulk sender: Developed using NextJs.
  • Bulk sender smart contracts: Solidity smart contracts for facilitating airdrops.
